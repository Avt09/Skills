# Layout Guide

Recommended one-page structure:

## Header

- Title: `MOEX Daily Market Review`
- Subtitle: session date

## Summary block

- 2 to 4 short lines based on the `overview` field

## Charts block

- Chart 1: top gainers
- Chart 2: top losers

Recommended chart styling:

- gainers: green palette
- losers: red palette
- ticker labels visible on every bar
- percentage labels shown near bar ends

## Footer

- compact source block
- optional note about liquidity metric

Design rules:

- favor clarity over decoration
- avoid multi-page layouts
- avoid dense tables
- keep whitespace around charts and labels

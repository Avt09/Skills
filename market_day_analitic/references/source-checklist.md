# Source Checklist

Use this checklist before finalizing the report.

## Market Data

- Confirm the exact trading session date.
- Use MOEX as the primary source for price change and liquidity.
- Prefer turnover in rubles as the liquidity filter when available.
- Confirm that all 10 securities belong to the liquid set you selected.

## Commentary

- Read only the pages that help explain the session.
- Prioritize BCS Express, Alfa Capital, Finam, and Aton in that order after MOEX.
- Do not cite a commentary source that does not add concrete context for the session.

## Accuracy

- Make sure the top gainers and losers come from the same session date.
- Check that percentage changes and tickers match the MOEX page.
- If the latest completed session is not the previous calendar day, say that explicitly.

## Output

- Include the exact session date in the opening sentence.
- Keep the overview short.
- Attach direct links to every page used.
